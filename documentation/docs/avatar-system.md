# Gestion d'avatar

- Création et personnalisation de créature virtuelle
- Évolution et suivi de l'état (faim, humeur, énergie...)
- Affichage dynamique de l'avatar dans l'application
- Interaction avec l'avatar (soins, jeux, etc.)
